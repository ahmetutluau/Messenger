# Messenger
In this application, you can message in realtime and send pictures, videos, locations as well as messaging.

⚠️ still in development ⚠️

## Tech Stack
- MessageKit
- Firebase Realtime
- SDWebImage
- JGProgressHUd
- Programmatic UI and Storyboards
- MVVM Architectural Pattern
- UIKit
